Sua Jornada para se Tornar um Desenvolvedor Front-end

Prepare-se para aprender HTML e CSS do zero e conquistar sua primeira vaga na área!

Módulo 1: Fundamentos do HTML

Aqui você vai aprender a criar a estrutura de qualquer página web. Vamos começar com o básico e evoluir até você dominar completamente o HTML.

  • Como organizar um documento web do zero
  • Criar títulos, parágrafos, listas e links
  • Adicionar imagens e criar formulários de contato
  • Usar as tags certas para melhorar seu código (HTML Semântico)

Módulo 2: Estilização com CSS

Agora vamos transformar suas páginas em algo visualmente atraente! Você aprenderá a dar vida ao HTML com cores, fontes e layouts profissionais.

  • Aplicar estilos e personalizar elementos da página
  • Trabalhar com cores e tipografia
  • Controlar espaçamentos e tamanhos (Box Model)
  • Posicionar elementos na tela

Módulo 3: Layouts Modernos

Esta é a parte mais importante! Você dominará as técnicas que as empresas mais usam hoje para criar sites que funcionam perfeitamente em celular, tablet e desktop.

  • Flexbox: organizar elementos de forma flexível
  • CSS Grid: criar layouts complexos em grade
  • Design Responsivo: fazer seu site funcionar em qualquer tela

Módulo 4: Preparação Profissional

Além do código, você precisa conhecer as ferramentas e práticas que todo desenvolvedor usa no dia a dia de trabalho.

  • Usar editores de código profissionais (VS Code)
  • Aprender Git para trabalhar em equipe
  • Criar seu portfólio com projetos reais
  • Preparar-se para o próximo passo: JavaScript

O que você será capaz de fazer:

  • Criar sites e landing pages do zero
  • Desenvolver páginas responsivas que funcionam em todos os dispositivos
  • Montar um portfólio profissional
  • Candidatar-se a vagas de Front-end Júnior
